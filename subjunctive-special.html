特殊な仮定法表現

I wish / if only / as if / without など、仮定法を使った慣用的表現を整理する。実際の会話・文章で頻繁に登場する重要パターン。

✨
Special Subjunctive Expressions
仮定法の基本形（if節）以外のパターン。日常英語・フォーマル文章に多数登場する。
I wish If only as if / as though Without / But for
01 I wish

I wish + 仮定法過去（現在への願望）

「〜ならよかったのに」「〜であればよいのに」と現在の状況に不満・願望を表す。wish の後ろに仮定法過去を使う。

I wish
願望
I could fly
仮定法過去（could + 原形）
飛べたらいいのになあ。（実際には飛べない）
現在への願望
I wish + S + 過去形
I wish I were taller. 背が高ければなあ。（今は低い）
She wishes she had a car. 車があればいいのに。（持っていない）
よく使う表現
I wish I knew the answer. 答えを知っていれば。（知らない）
I wish you were here. あなたがここにいてくれたら。
I wish it would stop raining. 雨が止んでくれればいいのに。（変化への願望）
would を使う場合： I wish + would は、主語の意志や変化への願望・軽い苛立ちを表す。
例: I wish he would stop talking. （彼が話すのをやめてくれればいいのに）
02 I wish + 過去完了

I wish + 仮定法過去完了（過去への後悔）

「〜していたらよかったのに」と過去の行動への後悔を表す。wish の後ろに仮定法過去完了（had + 過去分詞）を使う。

I wish
後悔
I had taken that opportunity
仮定法過去完了（had + pp）
あのチャンスを掴んでいればよかったのに。（実際は逃した）
過去への後悔
I wish + S + had + pp
I wish I had studied harder. もっと勉強していればよかった。
I wish she had told me sooner. もっと早く教えてくれればよかった。
よく使う表現
I wish I hadn't said that. あんなこと言わなければよかった。
I wish we had met earlier. もっと早く出会えていれば。
I wish he had been more careful. もっと慎重だったらよかったのに。
03 as if / as though

まるで〜のように

「まるで〜のように」と事実に反する様子を表す。as if / as though の後ろに仮定法を使う。現在の反実 → 過去形、過去の反実 → 過去完了形。

He talks
主節
as if he knew everything
as if + 仮定法過去
まるで何でも知っているかのように話す。（実際は知らない）
She acts as if she were the boss.
She acts as if she were the boss
まるでボスであるかのように振る舞う。（実際はボスでない）
種類例文意味
現在の反実
（as if + 過去形）		 He looks as if he were sick. まるで病気のように見える。（実際は違うかも）
過去の反実
（as if + 過去完了）		 She talks as if she had been there. まるでそこにいたかのように話す。（実際はいなかった）
as though
（= as if）		 He stared at me as though he had seen a ghost. 幽霊を見たかのように私を見つめた。
口語の as if： 実際には直説法（as if he knows everything）も口語では使われるが、正式には仮定法を使う。
04 If only

〜さえあれば / 〜だったらよかったのに

I wish と同じく仮定法を使うが、より強い感情・切望を表す。過去形 → 現在への願望、過去完了 → 過去への後悔。

If only
強い願望・後悔
I could go back in time!
仮定法過去
時間を遡れたらなあ！（強い願望）

If only + 過去形（現在）

If only I had more money!
お金がもっとあればなあ！
If only she were here now.
今ここに彼女がいてくれたら。

If only + 過去完了（過去）

If only I had known!
知っていたらよかったのに！
If only he hadn't left.
彼が去らなければよかったのに。
I wish vs If only： 意味はほぼ同じだが、If only のほうが感情が強く、しばしばエクスクラメーション（!）と一緒に使われる。
05 Without / But for

〜がなければ / 〜がなかったなら

Without / But for は「〜がなければ」という仮定を作る前置詞句。if節の代わりに使え、フォーマルな文体でよく使われる。

Without your help
Without + 名詞（If節の代わり）
,
I would have failed
主節（仮定法過去完了）
あなたの助けがなければ、失敗していただろう。
表現例文同義の if 文
Without + 名詞 Without water, nothing could live. → If there were no water …
水がなければ、何も生きられない。
Without + 名詞 Without your advice, I would have made a mistake. → If it had not been for your advice …
あなたのアドバイスがなければ、ミスしていただろう。
But for + 名詞 But for the rain, we could have played. → If it had not been for the rain …
雨がなければ、遊べたのに。
If it were not for If it were not for music, life would be dull. 音楽がなければ、人生は退屈だろう。（現在）
If it had not been for If it had not been for his effort, we would have lost. 彼の努力がなければ、負けていただろう。（過去）
まとめ： Without / But for は前置詞句なので動詞の形に気をつけなくてよいのが便利。
現在の反実 → 主節に would + 原形、過去の反実 → 主節に would have + pp。
