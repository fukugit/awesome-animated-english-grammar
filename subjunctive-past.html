仮定法過去

現在の事実に反する仮定を表す構文。「もし〜なら（実際はそうでないが）…だろうに」という反実仮想を、過去形を使って表現する。

🌀
Contrary-to-fact in the Present
現在の事実に反する仮定。過去形を用いることで「現実とはズレた世界」を表現する。
If + 過去形 would / could / might + 原形 were（be動詞） 反実仮想
基本公式
If S + 過去形,
S + would + 原形
01 基本構造

形と意味

If節に動詞の過去形を使い、主節に「would/could/might + 動詞原形」を置く。現在・未来の事実に反する仮定を表す。

If I were rich
IF節（過去形）
,
I would travel the world
主節（would + 原形）
もし私がお金持ちなら（実際はそうでない）、世界中を旅するだろうに。
ポイント： If節が過去形でも、意味は現在のこと。「今の事実に反する」ことを過去形で表すのが仮定法の約束事。
英文訳実際の事実
If I had more time, I would read more books.時間がもっとあれば、もっと本を読むのに。→ 実際は時間がない
If she knew your address, she would visit you.彼女があなたの住所を知っていれば、訪ねてくるのに。→ 実際は知らない
If it were sunny, we could go out.晴れていれば、外出できるのに。→ 実際は晴れていない
If he lived nearby, we might see him more often.彼が近くに住んでいれば、もっと会えるかもしれないのに。→ 実際は遠くに住んでいる
02 be動詞

were の使い方

仮定法過去でbe動詞を使う場合、主語が何であっても were を使うのが原則。口語では was も許容されるが、書き言葉や試験では were が正式。

If I were you
主語 I でも were
,
I would apologize
主節
私があなたなら（実際はそうでないが）、謝るでしょう。

✅ 仮定法では were

If I were …私が〜なら
If he were …彼が〜なら
If she were …彼女が〜なら
If it were …それが〜なら

慣用表現

If I were youあなたの立場なら
If I were in your shoesあなたの立場なら
If it were possible可能なら
Were I to …もし〜するとすれば（倒置）
注意： "If I was you" は口語では聞かれるが、仮定法の正式な形は were。試験・作文では were を使う。
03 直説法 vs 仮定法

現実と仮定の対比

直説法（普通のif文）は実現可能な条件を表す。仮定法過去は現在の事実に反することを表す。形の違いに注目。

直説法（条件文）
If it rains tomorrow, I will stay home.
明日雨なら（可能性あり）、家にいます。
仮定法（反実仮想）
If it rained now, I would stay home.
今雨だったら（実際は晴れ）、家にいるのに。
直説法
If you have time, please help me.
時間があれば手伝ってください。（普通の依頼）
仮定法
If you had time, you could help me.
時間があれば（実際はない）手伝えるのに。
見分け方： 動詞が現在形 → 直説法（実現可能な条件）。動詞が過去形 + 主節に would/could/might → 仮定法（反実仮想）。
04 助動詞の選択

would / could / might

主節の助動詞は would だけでなく could・might も使える。ニュアンスの違いを押さえておく。

would
〜だろうに（意志・確信）
If I had wings, I would fly to you. 翼があれば、あなたのところへ飛んで行くのに。
could
〜できるのに（能力・可能性）
If I were taller, I could play basketball. もっと背が高ければ、バスケができるのに。
might
〜かもしれないのに（弱い可能性）
If he tried, he might succeed. 挑戦すれば、成功するかもしれないのに。
まとめ： would（〜だろうに）> could（〜できるのに）> might（〜かもしれないのに）の順に確信度が下がる。
05 応用

If なし・倒置・Without

仮定法は if を使わずに表現することもできる。代表的なパターンを確認する。

Were I a bird
倒置（= If I were a bird）
,
I would fly to you
主節
もし私が鳥なら、あなたのところへ飛んで行くのに。
パターン例文意味
Were S + … Were I free, I would join you. 暇なら（= If I were free）一緒に行くのに。
Without + 名詞 Without your help, I would fail. あなたの助けがなければ、失敗するだろう。
But for + 名詞 But for the rain, we could play. 雨でなければ、遊べるのに。
to不定詞句 To hear him speak, you would think he's British. 彼が話すのを聞けば、イギリス人だと思うだろう。
倒置のポイント： Were / Had / Should で始まる倒置形は書き言葉・フォーマルな文体でよく使われる。If を使った形と同じ意味。
