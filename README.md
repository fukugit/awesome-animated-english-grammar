> Inspired by [dev-motion-cheats](https://github.com/fukugit/dev-motion-cheats) — same clean design philosophy, applied to English grammar.

**HTML / CSS / Vanilla JS — no external dependencies**

## 🎨 Design

- Clean minimal design — same token as dev-motion-cheats
- Consistent **color coding** across all pages:
  - 🔵 Blue — **S** Subject (主語)
  - 🟠 Orange — **V** Verb (動詞)
  - 🟢 Green — **O** Object (目的語)
  - 🟣 Purple — **C** Complement (補語)
  - 🔴 Red — **IO** Indirect Object (間接目的語)
- CSS animations triggered by IntersectionObserver (plays when scrolled into view)
- Replay buttons on every demo
- Fully responsive — works on mobile

## 🚀 GitHub Pages の設定方法

1. リポジトリの **Settings** → **Pages** を開く
2. Source を `Deploy from a branch` に設定
3. Branch を `main` / `/(root)` に設定
4. Save → 数分後に公開される
